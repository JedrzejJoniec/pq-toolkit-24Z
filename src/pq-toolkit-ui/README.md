# pq-toolkit-ui

## Development:

- Running development server:

```bash
npm run dev
```

## Deployment

### Requirements

- Docker
- Make

### Development environment

```
make build-dev
make start-dev
```

Open http://localhost

### Staging environment

```
make build-stage
make start-stage
```

Open http://localhost

### Production environment

```
make build-prod
make start-prod
```

Open http://localhost
